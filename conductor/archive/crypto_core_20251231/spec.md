# Track Spec: 建立核心顯示與 Wi-Fi 連線功能

## 1. 目的
本軌道旨在建立 Pico Crypto Tracker 的基礎設施，包含硬體初始化、網路連線以及最基礎的資料獲取與顯示流程。

## 2. 需求清單
*   **硬體初始化：** 正確驅動 Pico Display Pack 2.8 的 320x240 螢幕。
*   **Wi-Fi 連線：** 建立一個可靠的連線模組，支援 SSID/Password 設定並能回傳連線狀態。
*   **API 串接：** 使用 `urequests` 從公共 API（如 CoinGecko）獲取 BTC 目前價格。
*   **基礎 UI：** 在暗黑模式背景下，以大字體顯示 "BTC" 及其價格，並顯示連線圖示。

## 3. 驗證標準
*   螢幕能正確點亮並顯示文字。
*   裝置能成功連接指定的 Wi-Fi 並獲得 IP。
*   API 能正確回傳 JSON 並成功解析出價格數值。
*   螢幕上的價格能定期更新（如每分鐘一次）。
