# Track Spec: 互動式幣種與頻率切換功能

## 1. 概觀 (Overview)
提升 Pico Crypto Tracker 的互動性，允許使用者透過實體按鈕切換觀看的虛擬貨幣（BTC/ETH）以及自定義資料更新的頻率。

## 2. 功能需求 (Functional Requirements)
*   **按鈕 A 互動：** 循環切換顯示的幣種。
    *   路徑：`BTC` -> `ETH` -> `BTC`。
    *   行為：按下後更新目標幣種，待下一個更新週期獲取新資料。
*   **按鈕 B 互動：** 循環切換資料更新頻率。
    *   路徑：`10s` -> `15s` -> `30s` -> `60s` -> `10s`。
    *   行為：立即更改更新間隔計時器。
*   **介面顯示 (UI)：**
    *   在螢幕底部增加「資訊欄」。
    *   顯示目前選定的更新頻率與狀態。
*   **邏輯處理：** 
    *   實作按鈕去抖動 (Debouncing) 以避免重複觸發。
    *   非同步監聽按鈕輸入。

## 3. 非功能需求 (Non-Functional Requirements)
*   **回應速度：** 按鈕按下後，底部資訊欄應立即反應。
*   **程式碼結構：** 保持模組化，方便未來擴充更多幣種或頻率。

## 4. 驗收標準 (Acceptance Criteria)
*   按下按鈕 A 時，底部資訊欄顯示的目標幣種發生變更。
*   按下按鈕 B 時，底部資訊欄顯示的更新秒數依序循環切換。
*   下一次 API 更新會正確根據新選定的幣種獲取資料。

## 5. 範疇外 (Out of Scope)
*   在此軌道中不處理斷電後的設定儲存（設定將在重啟後恢復預設值）。
*   暫不支援 BTC/ETH 以外的幣種。
