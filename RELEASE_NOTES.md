# Release Notes

## v1.0.0 (2025-12-31)

### Initial Release
The first functional version of **Pico-Crypto-Tracker**.

#### Features:
- **Hardware Support:** Optimized for Raspberry Pi Pico 2 W and Pimoroni Display Pack 2.8".
- **Live Data:** Fetches real-time BTC/ETH prices from CoinGecko API via Wi-Fi.
- **Interactive UI:** 
  - Modern Dark Mode interface.
  - Button A: Cycle between BTC and ETH.
  - Button B: Cycle update intervals (10s, 15s, 30s, 60s).
- **System Feedback:** Integrated status bar with connection indicator and countdown timer.
- **Safety:** Automatic 429 Error (Rate Limit) protection.
